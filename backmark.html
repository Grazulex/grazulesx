Pourquoi BackMark ?

Les outils actuels n'ont pas été conçus pour le développement assisté par IA

❌ Le problème

  • Les task managers traditionnels (Jira, Linear, Notion) imposent leur UI et leur format propriétaire
  • L'IA ne peut pas collaborer naturellement dans ces outils fermés
  • Vos données sont enfermées dans des silos cloud
  • Performance médiocre avec de gros projets (1000+ tâches)
  • Complexité inutile pour des workflows de développement

✅ La solution BackMark

  • 100% Markdown + YAML - Format universel, éditable partout, versionnable avec Git
  • 4 AI Spaces dédiés - Planning, Notes, Documentation, Review pour vos assistants IA
  • Zéro lock-in - Vos fichiers restent lisibles même sans BackMark
  • Ultra-rapide - Indexation LokiJS : 50-250x plus rapide que la concurrence
  • CLI-first - Workflow développeur, scriptable, automatisable

Fonctionnalités

Tout ce dont vous avez besoin pour gérer vos tâches en collaboration avec l'IA

🤖

Design centré sur l'IA

BackMark reconnaît l'IA comme un membre d'équipe de première classe. Quatre espaces dédiés permettent à vos assistants IA de collaborer naturellement :

  • AI Planning - L'IA décompose les tâches complexes
  • AI Notes - Capture d'insights pendant le coding
  • AI Documentation - Documentation technique générée
  • AI Review - Feedback et suggestions de code
📝

Zéro Lock-in

Vos tâches sont stockées en Markdown pur avec frontmatter YAML. Pas de format propriétaire, pas de base de données opaque :

  • Lisible et éditable dans n'importe quel éditeur de texte
  • Versionnable avec Git - historique complet de vos tâches
  • Compatible avec tous les outils Markdown existants
  • Migration facile vers/depuis n'importe quel système
⚡

Performance extrême

Grâce à l'indexation LokiJS, BackMark offre des performances exceptionnelles :

  • 50-250x plus rapide que les task managers traditionnels
  • Gère 1000+ tâches sans ralentissement
  • Temps de réponse < 10ms pour les requêtes
  • Recherche floue instantanée sur tous les champs
🎨

Interface CLI moderne

Une expérience CLI élégante et productive pour les développeurs :

  • Affichage coloré et tableau Kanban interactif
  • Recherche floue et filtres puissants
  • Auto-complétion et invites intelligentes
  • Export JSON, CSV, Markdown pour scripting
🔒

Confidentialité totale

Vos données vous appartiennent. Point final.

  • 100% offline - Aucune connexion cloud requise
  • Zéro télémétrie - Aucune collecte de données
  • Pas de compte, pas d'authentification
  • Contrôle total sur vos fichiers
🔧

Automatisation IA

Laissez l'IA vous assister dans la gestion de vos tâches :

  • Décomposition automatique de tâches complexes
  • Estimation de complexité basée sur l'analyse du code

Démarrer avec BackMark

Installation et utilisation en 3 étapes

1

Installation

# Via npm (recommandé)
npm install -g backmark

# Via yarn
yarn global add backmark

# Via pnpm
pnpm add -g backmark
2

Initialisation

# Initialiser un nouveau projet
backmark init

# Créer votre première tâche
backmark add "Implement user authentication"

# Voir vos tâches
backmark list
3

Collaboration IA

# L'IA décompose la tâche
backmark ai-plan 42

# L'IA génère la documentation
backmark ai-docs 42

# Review avec l'IA
backmark ai-review 42
